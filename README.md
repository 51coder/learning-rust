## 实践之路

### 安装 Rust

```sh
curl https://sh.rustup.rs -sSf | sh
```

### 安装开发工具

Visual Studio Code 下载后直接安装，[https://code.visualstudio.com/download](https://code.visualstudio.com/download)。
